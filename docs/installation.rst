Installation
============

py_vollib_vectorized requires Python 3.5+.

Via pip
-------------------------

.. code-block:: none

    pip install py_vollib_vectorized

From source
-------------------------

.. code-block:: none

    git clone https://github.com/marcdemers/py_vollib_vectorized
    cd py_vollib_vectorized
    python setup.py
